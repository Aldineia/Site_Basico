História do computador

  1. Origens e Conceitos Iniciais
    • Ábaco (cerca de 2.400 a.C): Primeiro instrumento de cálculo.
    • Máquinas mecânicas:
      • Pascalina (1642): Calculadora mecânica de Blaise Pascal.
      • Máquina de Leibniz (1673): Realizava as quatro operações básicas.
    • Máquina Analítica (1873): Projeto de Charles Baddage.
    • Ada Lovelace: Criou o primeiro algoritmo para a máquina de Babbage - Primeira programadora da história.


  2. Primeira Geração (1940-1956)
    • Computadores usavam válvulas eletrônicas.
    • Eram enormes, lentos, consumiam muita energia.
    • Exemplo: ENIAC (1946) - primeiro computador eletrônico de uso geral.


  3. Segunda Geração (1956 - 1964)
    • Uso de transistores (substituíram as válvulas).
    • Computadores menores, mais rápidos e mais confiáveis.
    • Início das linguagens de programação como COBOL e FORTRAN.


  4. Terceira Geração (1965 - 1971)
    • Introdução dos circuitos integrados (chips).
    • Computadores mais compactos, com maior capacidade.
    • Surgimento dos sistemas operacionais.
    • Surgem os computadores pessoais (PCs).

  5. Quarta Geração (1971 - hoje)
    • Aparecimento do microprocessador (CPU em um chip).
      • Apple 1 (1976), IBM PC (1981).
    • Internet começa a se popularizar nos anos 1990.
    • Crescimento da informática em todos os setores.


  6. Quinta Geração (presente e futuro)
    • Computadores com inteligência artificial e aprendizado de máquina.
    • Uso de computação em nuvem, realidade aumentada, computação quântica e IoT.
    • Dispositivos cada vez menores, mais rápidos e inteligentes.

Linguagem de Programação

Uma linguagem de programação é um conjunto de regras e símbolos usados para escrever instruções que dizem a um computador o que fazer.

Em Resumo:
  • É uma forma de comunicação entre humanos e computadores.
  • Serve para criar programas, sistemas, sites, aplicativos, jogos, etc.
  • Transforma lógica e ideias em código que o computador entende.
Exemplos de linguagens de programação:
  • Python -> muito usada em ciência de dados, IA e automação.
  • JavaScript -> usada principalmente para páginas e aplicações web.
  • Java -> comum em sistemas corporativos e aplicativos Android.
  • C e C++ -> usada em sistemas operacionais e software de alto desempenho.

HTML é uma linguagem de programação?

O HTML (HyperText Markup Language) não é uma linguagem de programação, e sim uma linguagem de marcação.

O que isso significa:
  • O HTML define a estrutura e o conteúdo de uma página da web.
  • Ele organiza textos, imagens, links, vídeos e outros elementos para que o navegador saiba como exibir a página.
  • Funciona em conjunto com outras tecnologias:
    • CSS -> define o estilo (cores, fontes, layout).
    • JavaScript -> adiciona interatividade e lógica.

O HTML é a base da web, responsável por estruturar o que vemos em uma página.

Linguagem Server-side

Voltar

É a linguagem que roda no servidor de um site ou sistema.
Ou seja, quando você acessa uma página na internet:

  1. Seu navegador (cliente) faz um pedido para o servidor
  2. O servidor processa esse pedido usando uma linguagem server-side
  3. Ele consulta banco de dados, aplica regras, gera o conteúdo
  4. Só então envia o resultado pronto para o navegador

Veja mais: Wikipédia(5)

Linguagem Client-side

  • Linguagem Client-side -> é a que roda no navegador do usuário.
  • Função -> controla o que o usuário vê e interage na página.
  • Principais linguagens:
    • HTML -> estrutura
    • CSS -> aparência/ estilo
    • JavaScript -> interatividade

Entendendo listas ordenadas e não ordenadas

Voltar
  • Listas HTML permitem que desenvolvedores web agrupem um conjunto de itens relacionados em listas.
  • Uma lista ordenada começa com a (ol) tag.
  • Cada item da lista começa com a (li) tag.
  • Os itens da lista serão marcados com números por padrão.
  • Uma lista não ordenada começa com a (ul) tag.
  • Cada item da lista começa com a (li) tag. Os itens da lista serão marcados com marcadores (pequenos círculos pretos) por padrão.
